# Share without getting piled on

Short rules so launches feel like a person sharing a tool, not a campaign.

## Cadence

| Day | One action |
|---|---|
| Day 0 | Show HN **or** one list PR — not both in the same hour |
| Day 1–2 | Answer comments only |
| Day 3+ | Optional: one list PR (awesome-tmux) |
| Next week | Different product or different channel |

Never post the same block to HN + Reddit + three Discords in one morning.

## Voice

- First person is fine: “I built…”, “I kept hitting…”
- Author name: yours. No model credits.
- One install line. One repo link. Optional site.
- Admit limits before someone else does.

## Channels

| OK | Risky |
|---|---|
| Show HN (one tool) | 8 subreddits same text |
| One awesome-* PR | “Please star my fleet” |
| Reply in threads where someone has the problem | Cold DMs “check my SaaS” |
| Honest “limitations” section | Fake benchmarks |

## If someone is hostile

- One reply max, factual.
- Don’t dunk, don’t lecture about etiquette.
- Walk away. HN/Reddit memory is short; drama isn’t worth it.
